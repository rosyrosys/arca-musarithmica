# Core package: zero dependencies.
#
# `arca` itself runs on a clean Python 3.10+ interpreter — the five-step
# pipeline, the CLI, the rule log, the MIDI/MusicXML writers, the
# evaluator, the discrete-diffusion sampler, and the reference n-gram
# are all implemented against standard-library primitives only. Install
# with:
#
#     pip install -r requirements.txt
#
# ...will install nothing, by design. The file exists so reviewers can
# confirm the zero-dependency claim.
